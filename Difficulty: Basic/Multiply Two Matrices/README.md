<h2><a href="https://www.geeksforgeeks.org/problems/multiply-matrices/1">Multiply Two Matrices</a></h2><h3>Difficulty Level : Difficulty: Basic</h3><hr><div class="problems_problem_content__Xm_eO" style="--text-color: var(--problem-text-color);"><p><span style="font-size: 18px;">Given two square matrices <strong>a[][]</strong> and <strong>b[][]</strong>, each of size <strong>n * n</strong>, return the matrix obtained by multiplying a and b.</span></p>
<p><strong><span style="font-size: 20px;">Examples:</span></strong></p>
<pre><span style="font-size: 18px;"><strong>Input: </strong></span><span style="font-size: 18px;">a = [[7, 8], [2, 9]], b = [[14, 5], [5, 18]]</span>
<span style="font-size: 18px;"><strong style="font-size: 18px;">Output: </strong><span style="font-size: 18px;">[[138, 179], [73, 172]]<br></span><span style="font-size: 14pt;"><strong>Explanation:</strong> Multiplying the rows of a with the columns of b gives the resulting matrix.</span></span></pre>
<pre><span style="font-size: 18px;"><strong>Input: </strong></span><span style="font-size: 18px;">a = [[17, 4], [17, 16]], b = [[9, 2], [7, 1]]</span>
